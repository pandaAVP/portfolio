The site is one single page (`index.html`). Drop a correctly-named image into this `images/` folder and it shows up automatically — no code editing. If an image isn't there yet, that spot shows a neutral placeholder; nothing breaks.

## The even-easier option (recommended)
Don't worry about exact names. Just:
1. Drop any photos into this folder with descriptive names (e.g. `me headshot.jpg`, `nepse presentation.png`).
2. Ask me: *"look at my images folder and place them."*
3. I'll view each photo, rename/place it correctly, optimize it, and tell you to push.

## Tips
- Portrait photo: vertical (4:5) works best for `portrait.jpg`.
- Keep files under ~500 KB so the site stays fast (I can compress any photo you send).
